# AI PC Controller (JARVIS)

Десктопное приложение для управления ПК с помощью AI: чат с нейросетью, контекст из файлов (как в Cursor), агент с инструментами (открытие приложений, чтение/правка файлов, команды CMD, Telegram и др.), генерация изображений и видео, мессенджеры, системный мониторинг, игры и персонализация.

## Запуск

```bash
python main.py
```

Требуется Python 3.10+ и зависимости (PyQt6, requests, python-dotenv и др. — см. установку в проекте).

## Переменные окружения (.env)

Создайте файл `.env` в корне проекта.

**Нейросеть (чат и агент):**

- `NEURAL_API_KEY` или `API_KEY` — API-ключ провайдера
- `NEURAL_BASE_URL` — базовый URL API (для OpenAI-совместимых)
- `NEURAL_PROVIDER` — `openai`, `anthropic`, `deepseek` или `ollama`
- `NEURAL_MODEL` — модель (по умолчанию зависит от провайдера)

Для Anthropic: `ANTHROPIC_API_KEY`, для DeepSeek: `DEEPSEEK_API_KEY`, для OpenAI: `OPENAI_API_KEY` (если не задан `NEURAL_API_KEY`).

**Ollama (локальный чат без ключа):**

- `OLLAMA_BASE_URL` — по умолчанию `http://localhost:11434`
- `OLLAMA_MODEL` — например `qwen2.5:14b-instruct`

**Генерация изображений и видео (Replicate):**

- `REPLICATE_API_TOKEN` — токен Replicate

**Локальная генерация изображений (Fooocus):**

- `FOOOCUS_PATH` — путь к папке Fooocus
- `LOCAL_IMAGE_API_URL` — URL API (по умолчанию `http://127.0.0.1:7865` при наличии FOOOCUS_PATH)

**Telegram (опционально):** переменные для бота и пользовательского клиента (см. код и документацию по Telegram API).

## Основные вкладки

- **Главная** — центральная команда, быстрые команды, статус системы, история и лог
- **Файлы** — проводник по проекту и профилю, открытие/удаление/новая папка
- **Веб** — открытие URL, поиск Google/YouTube
- **Мессенджеры** — Telegram (чаты, AI-команды по переписке)
- **Игровой центр** — каталог игр, рекомендации
- **Система** — процессы, автозагрузка, системные действия
- **Автоматизация** — сценарии, AI-предложения
- **Аналитика** — статистика команд, мессенджеров и др.
- **Персонализация** — темы, настройки команд и AI
- **AI Чат** — чат с нейросетью, контекст @файл/@папка (как в Cursor), агент с инструментами, /clear, /image, /video, /help

В AI-чате: кнопка «@ Файл» добавляет выбранный на вкладке «Файлы» файл в контекст. При включённом подтверждении действий агента для правок в файлах показывается превью (diff) перед применением.
